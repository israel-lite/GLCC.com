# Life Builder's City Church - Children's Ministry

## 🌟 Welcome to Our Children's Ministry Website

A modern, responsive website for Life Builder's City Church Children's Ministry in Jos, Nigeria. Where faith meets fun!

## 🚀 Features

- **🏠 Modern Homepage** with welcome section, children activities, and video highlights
- **👥 Leaders Section** showcasing our dedicated ministers and teachers
- **🖼️ Interactive Gallery** with photos and videos of our activities
- **📞 Contact Page** with interactive map and contact information
- **📱 Mobile Responsive** design that works on all devices
- **✨ Beautiful Animations** using Framer Motion
- **🎨 Modern UI** with Tailwind CSS

## 🛠️ Technology Stack

- **React 18** - Frontend framework
- **React Router** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **PostCSS** - CSS processing

## 📱 Responsive Design

- 📱 Mobile-first approach
- 💻 Desktop optimized layouts
- 🎯 Touch-friendly interactions
- ♿ Accessibility considerations

## 🌐 SEO Optimized

- 🔍 Meta tags for search engines

## 🚀 Getting Started

### Prerequisites
- Node.js 14+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone [repository-url]

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

### Development

```bash
# Start local development server
npm start
# Runs on http://localhost:3000

# Run tests
npm test

# Build production bundle
npm run build
```
